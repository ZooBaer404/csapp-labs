## Setup
1. Download docker image
```
docker pull xieguochao/csapp
```

2. Run docker container and open web server
```
docker run -p 7777:7777 -v "$PWD/labs:/home/csapp/project" csapp
```

3. Copy repo inside ``/home/csapp/project``
```
git clone https://github.com/bmadone/csapp.git
```
